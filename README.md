# OOO Calendar

Aplikacja do rezerwacji dni wolnych stworzona w Streamlit.

## Funkcje
- Autoryzacja hasłem: `dexflex67`
- Wybór użytkownika (Michał, Dagmara, Julka, Szymon, Darek)
- Blokada zajętych dni
- System próśb o zamianę dni (🍩)
- Czarny motyw UI
- Baza danych SQLite

## Instalacja
```bash
pip install -r requirements.txt
streamlit run app.py
```
